# Large System Decoder - Site

## Estrutura do Projeto

```
lsd_site/
├── index.html          # Página principal
├── style.css           # Estilos do site
├── images/
│   ├── logo-gold.png   # Logo dourada (header esquerdo)
│   ├── logo-silver.png # Logo prateada (header direito)
│   ├── hero.jpg        # Imagem do painel principal
│   ├── card1.png       # Card 1 - Decodificação de Sistemas
│   ├── card2.png       # Card 2 - Construção Coletiva
│   └── card3.png       # Card 3 - Futuro é Feito de Código
```

## Como Usar

1. Copie todos os arquivos para uma pasta
2. Coloque as imagens na pasta `images/`
3. Abra `index.html` no navegador

## Imagens Necessárias

- **logo-gold.png**: Use `2.png` (logo dourada)
- **logo-silver.png**: Use `2.2.png` (logo prateada)
- **hero.jpg**: Use `lsd_real.png` ou `lsd_cyber.png` (painel principal)
- **card1.png**: Use `lsd_cyber.png` (card cyberpunk)
- **card2.png**: Use `lsd_real.png` (card realista)
- **card3.png**: Use `2049.png` (card futurista)

## Tema

O site segue um tema dark com dourado (#c9a227), inspirado nas logos do LSD.
