2. Import datetime: Explicação sobre importação e uso do módulo "datetime".

    - Exemplo 1 - imprimindo a data e hora agora: 

            import datetime

            x = datetime.datetime.now()
            print(x)

            >>> 2026-05-25 19:40:29.356703


    - Exemplo 2 - Imprimindo somente ano, ou somente mês:


            import datetime

            x = datetime.datetime.now()
            print(x.strftime("%Y")

            >>> 2026
