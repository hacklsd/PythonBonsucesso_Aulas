• ANALISANDO AS DIFERENÇAS ENTRE CLASS E ID: 


    Class pode ser atribuído a varios elementos, enquanto o Id somente a um. 
    
    
    Exemplo: 



    <!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

    <style>
        .container {
            width: 500px;
            height: 400px;
            background-color: azure;          
            text-align: center
            
        }

        #subcontainer{
            background-color: aquamarine;
        }

        #subcontainer p{   /* EDITANDO EXCLUSIVAMENTE O PARÁGRAFO DENTRO DA DIV. */
            font-size: 25px;
            color: brown;
            text-align: center;
            margin: auto;
        }
    </style>
</head>

<body style="background-color: antiquewhite">
    <div class="container">
        <div id="subcontainer">
            <p>Parágrafo na div (#subcontainer p) <hr></p>
        </div>
        <p> parágrafo fora da div (.container)</p>
    </div>
    
</body>
</html>
