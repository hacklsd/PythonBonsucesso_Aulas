GUIA DE AMBIENTE VIRTUAL PYTHON:


1)      python -m venv test_env             ---> Cria a pasta test_env com o ambiente isolado
2)      .\test_env\Scripts\Activate.ps1     ---> Ativa o ambiente virtual no PowerShell atual
3)      pip freeze                          ---> Lista todas as bibliotecas daquele módulo
4)      deactive                            ---> Desativa o ambiente e retorna ao ambiente global


    Um ambiente virtual, serve para:

    Executar bibliotecas específicas no desenvolvimento de um projeto, sem influenciar outros. 


------------------------------------------------------------------------------------------------------
Passo a passo para clonar e rodar um projeto do GutHub:




    • clonar projeto github: 

        -> git clone https:// link do github.git . (o espaço + . no final, é para colar todos os arquivos do projeto dentro da pasta)


    • Criar ambiente virtual:

        -> python -m venv nome_do_projeto


    • Ativação do ambiente virtual:

        -> .\nome_do_projeto\Scripts\Activate.ps1 (entrando pelo power shell, usar arquivo com extensão .ps1)


    • Instalação das bibliotecas do projeto: (Instalação dos arquivos requeridos pelo projeto (requirements.txt): Deve-se entrar na pasta do projeto *pelo POWER SHELL*: )

        -> pip install requirements.txt



Após esse passo a passo, o ambiente está pronto para testar/rodar o projeto clonado, de acordo com a documentação.
